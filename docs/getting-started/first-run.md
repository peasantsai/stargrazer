# First Run

## Starting the App

Launch Stargrazer. You'll see three views accessible from the sidebar:

- **Chat** — Upload content and interact with the browser
- **Schedules** — Manage automated jobs
- **Settings** — Connect accounts, configure browser, view logs

## Starting the Browser

Click **Start Browser** in the Chat header. This launches the bundled Chromium with:

- Stealth flags (hides automation signals)
- CDP enabled on port 9222
- Cookies extension pinned in the toolbar
- Dark mode forced
