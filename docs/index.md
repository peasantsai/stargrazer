# Stargrazer

**Social media automation with a bundled stealth browser.**

Stargrazer is a cross-platform desktop application that bundles an Ungoogled Chromium browser and controls it via the Chrome DevTools Protocol (CDP). Manage sessions, schedule uploads, and automate tasks across 6 social media platforms.

## Key Features

- :globe_with_meridians: **Bundled Chromium** — Ungoogled, stealth-configured, no telemetry
- :busts_in_silhouette: **6 Platforms** — Facebook, Instagram, TikTok, YouTube, LinkedIn, X
- :cookie: **Cookie Auth** — Paste Netscape cookies, sessions persist across restarts
- :alarm_clock: **Auto Keep-Alive** — Scheduled cookie refresh prevents session expiry
- :outbox_tray: **Content Uploads** — File + caption + hashtags to multiple platforms
- :calendar: **Job Scheduler** — Cron-based scheduling for all automated tasks
- :art: **Dark/Light Themes** — Toggle in the sidebar
